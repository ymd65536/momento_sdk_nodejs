# momento_sdk_nodejs

Momento SDKを使ってみる

## SetUp

 ```sh
npm install @gomomento/sdk
npm install dotenv
 ```

```sh
export MOMENTO_AUTH_TOKEN=<your Momento token here>
```

```sh
export MOMENTO_AUTH_TOKEN=
```

```sh
export MOMENTO_TTL_SECONDS=300
```
